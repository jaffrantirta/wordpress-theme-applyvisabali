# Apply Visa Bali - Quick Start (5 Minutes)

## 1. Install Theme (1 min)
```bash
# Upload the 'apply-visa-bali' folder to:
wp-content/themes/apply-visa-bali

# Then activate in WordPress Admin
Appearance > Themes > Apply Visa Bali > Activate
```

## 2. Set Homepage (1 min)
1. **Pages > Add New**: Create a page called "Home" (leave blank)
2. **Settings > Reading**:
   - Your homepage displays: **A static page**
   - Homepage: **Home**
   - Click **Save Changes**
