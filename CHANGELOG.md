# Apply Visa Bali Theme - Changelog

## Version 1.1.0 - 2026-01-05

### Added
- **Testimonials Section Customization**: Added full customizer controls for testimonials
  - 6 customizable testimonial slots
  - Customer name, photo, review text
  - Star rating selector (1-5 stars)
  - Optional position/company field
  - Shows 3 default testimonials with realistic sample content
  - Default testimonials: Sarah Johnson, Michael Chen, Emma Williams
  - Automatic avatar placeholder if no photo uploaded

- **What We Offer Section Customization**: Added customizer controls for services
  - 6 customizable service items
  - Icon (Bootstrap Icons) or custom image support
  - Title, description, and link for each service
  - Gradient background for icon-only cards

### Improved
- **Footer Widget Titles**: Fixed contrast issue with widget titles
  - All headings (h1-h6) now display in white
  - Widget titles have proper white color with !important
  - Improved text visibility on dark background
  - Better accessibility and readability

### Changed
- **Front Page**: Updated to use customizer options instead of post queries
  - Services section now pulls from customizer settings
  - Testimonials section now pulls from customizer settings
  - No longer requires creating posts or custom post types for homepage
  - Simplified setup process

### Documentation
- Updated INSTALLATION-GUIDE.md with new customizer sections
- Updated QUICK-START.md to reflect new configuration options
- Removed outdated instructions about creating service posts
- Added detailed instructions for testimonials customization
